ListContains ( LIST, VALUE )

Checks if a value is in a list.

Parameters

Parameter Type Description
LIST VALUE(LIST) The list to search.
VALUE VALUE The value to search for.

Returns

VALUE(BOOLEAN) TRUE if the value was found, or FALSE if not found.

What It Does

This function searches for the provided value in a list, and returns TRUE if found, or FALSE if not. If the provided list is not a list, FALSE is returned. The value can be any type, and strict equality is used to find the matching value to remove.

The search is done in sequential order.
