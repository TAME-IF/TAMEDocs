OnElementBrowse ( OBJECT-CONTAINER )

A block that describes what to do when an object is browsed when contained by a specific element or element parent.

Valid on Elements

OBJECT

Parameters

OBJECT-CONTAINER Any OBJECT-CONTAINER element type, including archetypes. The element type that this browse block is executed for when this object is contained by it.

When it is Called

The OnElementBrowse() block is called when TAME needs to execute a Browse() function, but only when the object that contains this block is in the browsed element. Several of these types of blocks may be called for one call of Browse(), but only the best matching one per object, maximum.

The OnElementBrowse() blocks are prioritized over the generic OnRoomBrowse(), OnPlayerBrowse(), OnContainerBrowse(), and OnWorldBrowse() blocks.

Inheritance rules apply when searching for callable blocks, always!

See Also

OnRoomBrowse() — Called when an object is browsed and owned by a room.
