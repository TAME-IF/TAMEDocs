RegexContains ( REGEX, STRING )

Checks if a string contains a pattern that matches the provided RegEx.

Parameters

Parameter Type Description
REGEX VALUE(STRING) The RegEx pattern to use.
STRING VALUE(STRING) The string to scan.

Returns

VALUE(BOOLEAN) TRUE if a match is found, FALSE if not.

What It Does

This function uses the provided RegEx pattern to find a matching substring in the provided input string. This returns TRUE if a matching substring is found, and FALSE if not.
