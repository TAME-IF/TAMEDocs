RegexSplit ( REGEX, STRING )

Splits a string into multiple substrings using a RegEx pattern as a delimiter.

Parameters

Parameter Type Description
REGEX VALUE(STRING) The RegEx pattern to use.
STRING VALUE(STRING) The string to split.

Returns

VALUE(LIST) The list of substrings returned after splitting.

What It Does

This function splits a string into a list of substrings by splitting it up at the boundaries of each match of the provided RegEx pattern. Both parameters are interpreted as strings before the split.
