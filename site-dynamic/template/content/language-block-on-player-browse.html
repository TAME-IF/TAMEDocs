OnPlayerBrowse ( )

A block that describes what to do when an object is browsed when contained by a player.

Valid on Elements

OBJECT

Parameters

No Parameters

When it is Called

The OnPlayerBrowse() block is called when TAME needs to execute a Browse() function with a player as a parameter and when the object that contains this block is in the browsed player. Several of these types of blocks may be called for one call of Browse().

Inheritance rules apply when searching for callable blocks, always!
