RemoveObjectTagFromAllIn ( OBJECTCONTAINER, TAG )

Removes a tag from each object in an object container.

Parameters

Parameter Type Description
OBJECTCONTAINER OBJECT-CONTAINER The object container element.
TAG VALUE(STRING) The tag to remove from each object.

Returns

Returns nothing. Cannot be used in expressions.

What It Does

This function removes a tag from each object that is owned by a particular object container element. Removing tags that are not assigned to the provided objects does nothing.

Unlike names, tags are not sanitized on assignment to objects - they are added as-is.

Tags are matched case-insensitively.
