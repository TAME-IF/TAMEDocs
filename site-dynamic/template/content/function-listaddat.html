ListAddAt ( LIST, ITEM, INDEX )

Adds an item at a specific index of a list.

Parameters

Parameter Type Description
LIST VALUE(LIST) The list to revome from.
ITEM VALUE The item to search for and remove.
INDEX VALUE(INTEGER) The index to add the value at.

Returns

VALUE(BOOLEAN) TRUE if added, FALSE if not.

What It Does

This function adds a value at a specific index in a list, and returns TRUE if added. If the provided list is not a list, or the index provided is less than 0, nothing is added and this returns FALSE. If the index is greater than or equal to the list's length, it is added to the end of the list. The item can be any type, and is added as-is without any conversion.

A successful add increases the list's size by 1.
