PushRoom ( PLAYER, NEWROOM )

Changes the current room on a specific player by adding ("pushing") a room on the top of the player's room stack.

Parameters

Parameter Type Description
PLAYER PLAYER The player whose room stack will change.
NEWROOM ROOM The room to add.

Returns

Returns nothing. Cannot be used in expressions.

What It Does

This function sets the topmost room in a specific player's room stack to the provided room by "pushing" the provided room onto the top of it (thus making it "current" for that player). From the moment that this function is executed, all functions and operations that use the current room (provided that its associated player is also current) will use the one provided until another call to PushRoom() (or the other room-changing functions) changes it again.

Since PushRoom() preserves the previously current room that the provided player was in, a call to PopRoom() will restore it for that player.
