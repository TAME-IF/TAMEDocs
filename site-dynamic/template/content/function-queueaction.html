QueueAction ( ACTION )

Enqueues a general action command to execute.

Parameters

ACTION A GENERAL action. The general action to encapsulate into a command and enqueue as the next command.

Returns

Returns nothing. Cannot be used in expressions.

What It Does

This function takes a general action and creates a command to execute after the current command completes. This does not break execution nor interrupt it.
