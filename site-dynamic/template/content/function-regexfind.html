RegexFind ( REGEX, STRING )

Finds the index of the first matching substring that matches the provided RegEx and returns it.

Parameters

Parameter Type Description
REGEX VALUE(STRING) The RegEx pattern to use.
STRING VALUE(STRING) The string to scan.

Returns

VALUE(INTEGER) The index of the first match or -1 if not found.

What It Does

This function uses the provided RegEx pattern to find the first matching substring in the provided input string. This returns the starting index of the match if a matching substring is found, and -1 if nothing was matched. Both parameters are interpreted as strings before the scan.
