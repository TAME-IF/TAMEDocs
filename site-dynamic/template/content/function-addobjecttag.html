AddObjectTag ( OBJECT, TAG )

Adds a tag to an object.

Parameters

Parameter Type Description
OBJECT OBJECT The object to add to.
TAG VALUE(STRING) The tag to add.

Returns

Returns nothing. Cannot be used in expressions.

What It Does

This function adds a tag to an object. Adding a tag that is already assigned to the provided object does nothing.

Unlike names, tags are not sanitized on assignment to objects - they are added as-is.

Tags are matched case-insensitively.
