OnUnknownCommand ( )

A block that describes what to do when a valid action cannot be parsed from the input.

Valid on Elements

WORLD PLAYER

Parameters

No Parameters

When it is Called

The OnUnknownCommand() block is called when the interpreter cannot parse a valid action from the interpreted input by name.

When TAME needs to execute this block, it searches for it on the current player, then the world.

Inheritance rules apply when searching for callable blocks, always!

Example

Additional Technical Notes

The OnUnknownCommand() block is context sensitive involving the current player. This is assessed when finding the relevant block to execute for the current action being processed. This will affect program flow when players change.
