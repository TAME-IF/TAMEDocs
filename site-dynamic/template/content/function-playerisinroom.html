PlayerIsInRoom ( PLAYER, ROOM )

Checks if a player is a particular room.

Parameters

Parameter Type Description
PLAYER PLAYER The player to check.
ROOM ROOM The room to inspect.

Returns

VALUE(BOOLEAN) TRUE if the player is in the room, FALSE if not.

What It Does

This function checks if a player is in a particular room, or rather, if the provided player's current room is the provided room. If so, this returns TRUE. If not, this returns FALSE.

This does not return TRUE if the provided room is a room in the player's room stack - only if it is the topmost one (a.k.a. the "current" one).

Unless there's no current player or no current room, the following should always return true: 


	PlayerIsInRoom(player, room);

Example

See Also

SetRoom() — Sets the current room.
PushRoom() — Pushes a room onto a player's room stack.
PopRoom() — Removes the topmost room from a player's room stack.
SwapRoom() — Changes the topmost room on a player's room stack.
