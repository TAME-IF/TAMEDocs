ListRemove ( LIST, ITEM )

Removes an item from a list.

Parameters

Parameter Type Description
LIST VALUE(LIST) The list to append to.
ITEM VALUE The item to remove.

Returns

VALUE(BOOLEAN) TRUE if removed, FALSE if not.

What It Does

This function removes a value from a list, and returns TRUE if matched and removed. If the provided list is not a list or the value is not found, nothing changes and this returns FALSE. The item can be any type, and strict equality is used to find the matching value to remove.

A successful remove decreases the list's size by 1.

The search is done in sequential order.
