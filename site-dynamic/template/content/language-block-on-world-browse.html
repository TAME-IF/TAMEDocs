OnWorldBrowse ( )

A block that describes what to do when an object is browsed when contained by a world.

Valid on Elements

OBJECT

Parameters

No Parameters

When it is Called

The OnWorldBrowse() block is called when TAME needs to execute a Browse() function with the world as a parameter and when the object that contains this block is in the world. Several of these types of blocks may be called for one call of Browse().

Inheritance rules apply when searching for callable blocks, always!
