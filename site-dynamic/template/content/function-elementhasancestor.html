ElementHasAncestor ( ELEMENT, PARENT )

This function checks if an element has another element as an ancestor.

Parameters

Parameter Type Description
ELEMENT ELEMENT Any element (archetype or non-archetype).
PARENT ELEMENT Any element to check the possible parentage of (archetype or non-archetype).

Returns

VALUE(BOOLEAN) TRUE if the first element is an ancestor of the second element, or FALSE otherwise.

What It Does

This function tests if an element has the other provided element as an ancestor (the first element is a descendant). This returns TRUE if the first element is an ancestor of the second element, or FALSE otherwise.
